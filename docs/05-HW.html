Homework exercise

Learning Objectives

  • Use publicly available RNA-seq data
  • Perform DE analysis with DESeq2

Task

There are many RNA-seq datasets available in the public domain. Find an RNA-seq dataset of interest to you and perform a DE analysis using DESeq2. You can use the code from the previous exercises as a template for your analysis.

You can search for datasets on GEO or SRA. You can also use datasets from the Bioconductor ExperimentHub.

The two datasets below are available as Bioconductor packages. Read the package vignettes for each dataset to learn more about the experimental design and how to access the data in R.

Airway dataset

The [airway]https://www.bioconductor.org/packages/release/data/experiment/html/airway.html) dataset contains RNA-seq data from human airway smooth muscle cells treated with dexamethasone (a synthetic glucocorticoid) or left untreated.

Pasilla dataset

The pasilla dataset contains RNA-seq data from Drosophila melanogaster cells treated with dsRNA against the splicing factor pasilla or left untreated.

Output

  • Quality control plots to assess quality and reproducibility of the data
  • A table of differentially expressed genes with log2 fold changes and adjusted p-values
  • A volcano plot to visualize the results of the DE analysis
  • A brief interpretation of the results, including any interesting genes or pathways that were identified
